ATRIBUTOS BASICOS EN HTML

Atributos id, class, style y title

ATRIBUTO ID

Establece un identificar único a cada elemento dentro de una página html

ATRIBUTO CLASS

Establece la clase CSS que se aplica a los estilos del elemento.
Si un elemento tiene es por ejemplo un título y también se le asignó una class, se le aplicarán las características de la class, siempre formateo por proximididad

ATRIBUTO STYLE

Establece de forma directa los estilos CSS indicados para una etiqueta concreta, aquella que lleve el atributo Style. Si quisiéramos aplicar eses formatos a todos los elementos de ese tipo mejor hacerlo con el atributo class

ATRIBUTO TITLE

Aparece un pequeño texto al poner el puntero encima de ese elemento. Aumenta la claridad de la web y facilita su navegación
